## 🚀 Quick Start
```bash
# Clone repository
git clone https://github.com/Lovingthemoo-74/SEO-Friendly-Personal-Portfolio-Website
cd SEO-Friendly-Personal-Portfolio-Website

# Install dependencies (if any)
npm install

# Run development server
npm run dev
```
